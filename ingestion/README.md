# RAG-Enhanced Framework for Dynamic Query Resolution in Infrastructure Inspection

## Overview

This project uses Retrieval-Augmented Generation (RAG) with Mistral LLM, PostgreSQL PGVector, and LangChain to provide intelligent responses for infrastructure inspection and compliance-related queries.

## Features

- PDF document ingestion
- Semantic search using vector embeddings
- PostgreSQL + PGVector storage
- Mistral-powered question answering
- Flask REST API

## Tech Stack

- Python
- LangChain
- Ollama
- Mistral
- PostgreSQL
- PGVector
- Flask
